# Easy Library

**Easy Library** is a powerful and dynamic asset management add-on for Blender, developed by **Jopex Creatives**. It allows you to effortlessly manage, preview, and import your `.blend`, `.fbx`, `.obj`, `.glb`, `.gltf`, `.exr`, and `.hdr` assets directly from the 3D Viewport sidebar.

## Features
- **Dynamic Asset Loading**: Automatically scans and displays supported 3D formats and HDRI textures from your library.
- **Native Thumbnails**: Displays internal `.blend` preview thumbnails automatically in a clean, symmetrical grid layout.
- **Toggleable Folders**: Organize your assets neatly with expandable/collapsible category folders.
- **Import Versatility**: 
  - Imports 3D formats automatically, parenting them to a scalable Empty object.
  - Automatically configures `.exr` and `.hdr` files as World Environment textures with full background shader nodes.
- **Robust Management UI**: Right from the panel, you can:
  - Add and Remove Folders
  - Import new assets into specific folders
  - Delete assets safely with double confirmation
  - Move assets between folders
  - Rename assets and folders
- **Live Search**: Instantly filter assets by name or category.

## Installation
1. Download the `Easy_Library` folder or ZIP archive.
2. In Blender, go to `Edit > Preferences > Add-ons`.
3. Click `Install...` and select the `__init__.py` or the downloaded `.zip`.
4. Enable the add-on by checking the box next to **Object: Easy Library**.
5. Ensure your asset files are placed within the `Assets` folder inside the add-on directory.

## Usage
- Open the **Sidebar** (`N` key) in the 3D Viewport and click on the **Easy Library** tab.
- Click any thumbnail or asset name to immediately add it to your scene.
- Use the **Management buttons** at the bottom to expand and organize your library.
